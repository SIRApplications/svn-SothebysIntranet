Module
======
Views Watchdog


Description
===========
This module extends the Views module and allows to create customized lists
(pages, blocks, feeds) of watchdog entries.


Requirements
============
- Drupal 6.x
- Views 6.x-2.x


Installation
============
1. Move this folder into your modules directory.
2. Enable it from Administer >> Site building >> Modules >> Views.


Configuration & Using
=====================
- Create customized lists at Administer >> Site building >> Views.
- Enable the default view at Administer >> Site building >> Views.
